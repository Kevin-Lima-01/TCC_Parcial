## Sobre o Trabalho

Este repositório contém o código-fonte e os dados necessários para replicar os resultados da monografia que investiga a inclinação da Curva de Phillips no Brasil utilizando variação regional entre municípios.

O trabalho replica e adapta para o Brasil a estratégia de identificação de Hazell et al. (2022), que explora variação regional dentro de uma união monetária para contornar dois problemas fundamentais:

1. **Viés de expectativas** - deslocamentos nas expectativas de inflação de longo prazo se confundem com a inclinação estrutural κ
2. **Viés de simultaneidade** - neutralização de choques de demanda pela política monetária

### Principais Resultados

- Curva de Phillips negativa e significativa para bens **non-tradables** (ψ̂ = -0,097, p < 0,05; κ̂ = -0,009, p < 0,001)
- Curva estatisticamente nula para bens **tradables**, validando a estratégia de identificação regional
- O aparente achatamento da curva reflete principalmente o grau de ancoragem das expectativas de inflação de longo prazo
- Medida ampla de folga que incorpora informalidade domina o desemprego aberto na explicação da inflação de non-tradables
